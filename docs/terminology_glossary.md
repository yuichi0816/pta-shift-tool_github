# 用語集（tool / manual 共通）

作成日: 2026-02-07  
対象: `tool.html`, `manual.html`, `js/script.js` のユーザー表示文言

## 統一用語

| 用語 | 定義 | 備考 |
|---|---|---|
| 参加可能回数 | 各参加者が回答した「対応可能な回数」 | 主用語。`参加回数` は使わない |
| 参加確認 | 参加可否・免除意向の回答項目 | `参加可能回数` とは別項目 |
| 割当て回数 | 実際に割当てられた回数 | `割り当て回数` 表記は使わない |
| 未割当て | 埋まらなかった枠、または割当てられなかった参加者 | `未割り当て` 表記は使わない |
| 特定参加可能日 | 「この日なら参加できる日」 | `特定希望日` は使わない |

## 運用ルール

- 新規文言を追加するときは本用語集の語彙を優先する。
- 同一画面内で同義語を混在させない。
- `tool.html` と `manual.html` で同一概念は同じ語を使う。
